# 🍽️ Recipe Recommendation Chatbot

## Overview
The Recipe Recommendation Chatbot is a Python-based application that recommends recipes based on user input. The chatbot provides recipe suggestions and displays the ingredients required for the selected recipe through an interactive Gradio interface.

## Features
- Interactive chatbot interface
- Recipe recommendations
- Ingredient display
- User-friendly design
- Built using Python and Gradio

## Technologies Used
- Python
- Gradio
- Pandas
- Jupyter Notebook

## Project Structure

Recipe-Chatbot/
│
├── Recipe_Chatbot.ipynb
├── recipes.csv
├── README.md
└── requirements.txt

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required libraries.
3. Run all cells.
4. Click the generated Gradio link.
5. Enter a recipe name and receive recommendations.
